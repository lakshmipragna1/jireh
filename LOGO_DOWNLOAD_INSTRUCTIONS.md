## ðŸ“¥ How to Download & Save

### Method 1: Direct Download (Recommended)
1. Right-click on each logo image from the sources above
2. Select "Save image as..."
3. Navigate to `public/brands/` folder in your project
4. Save with the **exact filename** listed above (e.g., `alg-chemicals.png`)

### Method 2: Screenshot & Crop
1. Take a screenshot of the logo
2. Crop to remove background and extra space
3. Save as PNG with transparent background (if possible)
4. Save to `public/brands/` with the correct filename

### Method 3: Google Image Search
1. Search for "[Brand Name] logo PNG transparent"
2. Look for high-quality images (at least 200px wide)
3. Download and save to `public/brands/` folder
